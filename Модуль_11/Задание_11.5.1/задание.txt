Задание_11.5.1

1. Установить на свой компьютер интерпретатор Python последней версии
2. Создать два проекта, для каждого создать новое виртуальное окружение.
3. Установить в каждое окружение по 2 пакета, отличных от тех, которые использовались ранее. Причём один пакет в каждом окружении должен отличаться только версией. Для этого вам самостоятельно нужно найти, как с помощью модуля pip установить пакет, с указанием определённой версии.
4. С помощью команды pip freeze сравнить результат одного и второго виртуального окружения.
